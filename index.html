Buttons

Buttons can be applied to both anchor tags and button tags. Alternatively, the button module can be applied to an input tag. However, the loading animation will not be applied. Our buttons are fully responsive.

To set up the button, simply add the Button css class to your button, anchor or input tag. 

        
          <button class="Button">Button</button>
          <a class="Button">Button</a>
          <input class="Button" type="submit" value="Button" />

Result

Button

Width

By default, the button will take 100% of the width of its parent element. To prevent this, add Button--minWidth class to the element. This will force the button to take the minimum width with 20px padding on either side. 

        
          <button class="Button Button--minWidth">Button</button>

Result

Height

By default, the button will take 40px of the height (45px on mWeb). To prevent this, add Button--low class to the element. This will force the button to take the height 30px (30px on mWeb). 

        
          <button class="Button Button--low">Button</button>
          <button class="Button Button--minWidth Button--low">Button</button>

Result

Icons

Some buttons may contain icons. The ideal source for these icons is FontAwesome. You can add icons to any button. The maximum icon size for button should be 12px. 

        
          <button class="Button Button--minWidth">
            <i class="icon-star-empty" aria-hidden="true"></i> Button
          </button>
          <a class="Button Button--hollow">
            <i class="icon-star-empty" aria-hidden="true"></i> Button
          </a>
Button

Disabled

All buttons have a default state for disabled. If Button is being applied to an anchor tag, adding the class isDisabled will give it the same styling as a disabled button. There is an alternate hollow version of the disabled button. To use this style, add Button--hollowDisabled to the button. Then, when the button is disabled, the hollow style will be applied. In this case, the color of the enabled button does not affect its disabled style. 

        
          <button class="Button Button--minWidth" disabled>Button</button>
          <a class="Button Button--ink Button--minWidth isDisabled">Button</a>
          <button class="Button Button--classicBlue Button--minWidth Button--hollowDisabled" disabled>Button</button>
          <a class="Button Button--highlightTeal Button--minWidth Button--hollowDisabled isDisabled">Button</a>

Result

Button Button

Loading

With the exception of the input tag, all buttons can have a loading animation attached to them. Adding the isLoading class to the button will apply the loading animation. Below is an example of how isLoading is being applied to the buttons. The code below also removes the animation after it is finished (3 seconds) for the sake of reusability. In reality, the load animation does NOT go away until isLoading is removed; Ideally, you would not want to continuously add and remove isLoading. 

        
          <script>
            (function(document) {
              var buttonClass = document.getElementsByClassName('jsButton');

              Array.from(buttonClass).forEach(function(element) {
                element.addEventListener('click', loadingButton);
              });

              function loadingButton() {
                var loadingTimeout;
                var e = this;
                if(!e.classList.contains('isLoading')) {
                  e.classList.add('isLoading');
                  loadingTimeout = setTimeout(function() {
                    e.classList.remove('isLoading');
                  }, 3000);
                }
              }
            })(document);
          </script>

          <button class="jsButton Button Button--minWidth">Button</button>
          <a class="jsButton Button Button--highlightTeal Button--minWidth">Button</a>
          <button class="jsButton Button Button--hollow">Button</button>

Form's elements

Inputs

By default, the input will take 100% of the width of its parent element. To prevent this, add TextField--minWidth  class to the element. For styling textarea add TextField--area  class to the element. This will force the button to take the minimum width. If the input should be part of the form without space between the fields you should add TextField--group  class to the element. 

        
          <input class="TextField TextField--minWidth u-margin-bottom-30" type="text" placeholder="Text here" />
          <input class="TextField TextField--minWidth isError  u-margin-bottom-30" type="text" placeholder="Text here" />

          <input class="TextField TextField--group" type="text" placeholder="Email" value="user@gmail.com" />
          <input class="TextField TextField--group isError" type="text" placeholder="Phone number" />
          <input class="TextField TextField--group" type="text" placeholder="Your name" />
          <textarea class="TextField TextField--area TextField--group" placeholder="Text here" rows="4" cols="50">
